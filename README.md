# Práctica LAMP con Vagrant (Debian 12 Bookworm)

Este proyecto crea automáticamente un entorno **LAMP** (Linux, Apache, MariaDB, PHP) usando **Vagrant** y **VirtualBox**.  
Consta de **dos máquinas vagrant** separadas:

**ricardofrutosbravodb** → Servidor de base de datos con **MariaDB**
**ricardofrutosbravoapache** → Servidor web con **Apache + PHP**

# Creditos
